# Routes-in-Ingushetia - Маршруты Ингушетии

Проблема: В городе нет единого источника информации о маршрутах маршрутных такси. Жителям и гостям приходится спрашивать номера маршруток, остановки и время отправления у других людей.

Решение: Сайт, на котором пользователь может ввести начальную и конечную точку поездки и получить список подходящих маршрутов с номерами, остановками и расписанием.

Как работает: Пользователь вводит откуда и куда нужно поехать. Система ищет совпадения по всем остановкам маршрута (включая промежуточные) и выводит подходящие варианты.

Инструменты: 
1. Структура страницы HTML
2. Внешний вид и стили CSS
3. Логика поиска и отображение данных JavaScript
